# DATE-TIME FUNCTIONS

- CURRENT_DATE
  ```select date() as date;```

- CURRENT_TIME
  ```select time() as time;```

- CURRENT_TIMESTAMP
  ```SELECT CURRENT_TIMESTAMP AS current_datetime;```
