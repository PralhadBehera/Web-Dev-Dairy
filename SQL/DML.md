# DML

- **INSERT** :
```sql
INSERT INTO table_name (column1, column2, ...)
VALUES (value1, value2, ...);
```

- **UPDATE** :
```sql
UPDATE table_name
SET column1 = value1, column2 = value2, ...
WHERE condition;
```
- **DELETE** :
  - To Delete specific record in table 
  ```sql 
  DELETE FROM table_name
  WHERE condition;|
  ```
  - To Delete entire record in table 
  ```sql
  DELETE FROM table_name
  ```
